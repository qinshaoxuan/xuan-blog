LeetCode 42 Trapping Rain Water

  • 单调栈

题目链接：Trapping Rain Water

题意：给n个非负数$$h_i$$，表示每节宽度为1的高程图，求下雨后能装下的最大水量

如0 1 0 2 1 0 1 3 2 1 2 1 答案为6
 

可用单调栈解决，栈内存下标，保证对应的$$h_i$$不严格递减

每次出栈时，出栈位置可贡献宽度为$$i-stack[top - 1]-1$$，高度为$$min(h_{stack[top-2]}\ ,h_i)-h_{stack[top]}$$的矩形区域

时间复杂度$$O(n)$$

 

public class Solution {
    public int trap(int[] height) {
        int ans = 0;
        int[] stack = new int[height.length];
        int top = 0;
        for (int i = 0; i < height.length; ++i) {
            while (top > 0 && height[stack[top - 1]] < height[i]) {
                if (top > 1)
                    ans += (i - stack[top - 2] - 1)

                        * (Math.min(height[stack[top - 2]], height[i]) - height[stack[top - 1]]);
            top--;
        }
        stack[top++] = i;
    }
    return ans;
}

}
