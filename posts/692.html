题目链接：String to Integer (atoi)

题意：将字符串转化成整数，实现C语言中的atoi()函数
这道题有很多细节：

  1. 去掉前导空格
  2. 首部必须为’+’,’-‘或者数字，否则返回0,首部有多个’+’,’-‘也返回0
  3. 空串或者单独的’+’,’-‘返回0
  4. 舍去额外的字符后面的串
  5. 如果数值超过int范围取int的极值
  6. 输入的字符串转化成整数可能会超出long范围
     
    public class Solution {
    public int myAtoi(String str) {
    str = str.trim();
if (str.isEmpty())
    return 0;
if (str.charAt(0) != '-' &amp;&amp; str.charAt(0) != '+' &amp;&amp; (str.charAt(0) &lt; '0' || str.charAt(0) &gt; '9'))
    return 0;
int op = (str.charAt(0) == '-') ? -1 : 1;
if (str.charAt(0) == '-' || str.charAt(0) == '+')
    str = str.substring(1);
long ans = 0;
int len = str.length();
for (int i = 0; i &lt; len; ++i) {
    if (str.charAt(i) &lt; '0' || str.charAt(i) &gt; '9')
        break;
    ans *= 10;
    ans += str.charAt(i) - '0';
    if (ans &gt; Integer.MAX_VALUE)
        break;
}
if (op == 1)
    return (int) Math.min(Integer.MAX_VALUE, ans * op);
else
    return (int) Math.max(Integer.MIN_VALUE, ans * op);

    }
    }
