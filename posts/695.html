题目链接：Regular Expression Matching

题意：实现关于’.’和’‘的正则表达式匹配，’.’匹配任何单一字符，’‘匹配0个或多个前一字符

给出字符串s和正则表达式p,判断是否完全匹配
 

‘.’的匹配很简单，判断下就行，主要是’*’的匹配

如果p的下一个字符不是*，那么就判断当前的字符是否匹配

如果p的下一个字符是*，那么就枚举前一个字符的数量，对于每种情况进行匹配。若有一个分支完成匹配，则当前串可以匹配。

 

public class Solution {
    public boolean judge(String s,String p,int i,int j){
        if (j == p.length())
            return i == s.length();
        if (j + 1 == p.length())
            return (i + 1 == s.length() && (s.charAt(i) == p.charAt(j) || p.charAt(j) == ‘.’));
        if (p.charAt(j + 1) != ‘*’) {
            if (i != s.length() && (s.charAt(i) == p.charAt(j) || p.charAt(j) == ‘.’)) {
                return judge(s, p, i + 1, j + 1);
            }
        } else {
            while (i != s.length() && (s.charAt(i) == p.charAt(j) || p.charAt(j) == ‘.’)) {
                if (judge(s, p, i, j + 2))
                    return true;
                i++;
            }
            return judge(s, p, i, j + 2);
        }
        return false;
    }
    public boolean isMatch(String s, String p) {
        return judge(s, p, 0, 0);
    }
}
