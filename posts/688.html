题目链接：Reverse Integer

题意：翻转数字
要注意符号，和翻转后的前导零。

还要判断翻转后是否超出int范围，超出输出0

JAVA的int范围为$$-2^{31} \sim 2^{31}-1$$

 

public class Solution {
    public int reverse(int x) {
        int op = (x >= 0) ? 1 : -1;
        x = Math.abs(x);
        long ans = 0;
        while (x != 0) {
            ans = 10;
            ans += x % 10;
            x /= 10;
        }
        if (ans  op > Integer.MAX_VALUE || ans  op < Integer.MIN_VALUE)
            return 0;
        else
            return (int) (ans)  op;
    }
}
