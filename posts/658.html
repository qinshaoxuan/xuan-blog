题目链接：Longest Substring Without Repeating Characters

题意：给出一个字符串，找出最长的无重复字符（只出现一次）的子串
 

可用二指针解决

当前子串满足条件时，右指针右移，更新答案，直到不满足条件。此时左指针右移直到满足条件。

重复以上过程直到右指针指向字符串末尾

时间复杂度$$O(n)$$

 

public class Solution {
    public int lengthOfLongestSubstring(String s) {
        int vis[] = new int[260];
        int ans = 0;
        int l = 0, r = 0;
        while (l <= r && r < s.length()) {
            vis[s.charAt(r)]++;
            if (vis[s.charAt(r)] > 1) {
                while (vis[s.charAt(r)] > 1) {
                    vis[s.charAt(l)]–;
                    l++;
                }
            }
            ans = Math.max(ans, r - l + 1);
            r++;
        }
        return ans;
    }

}
