LeetCode 6 ZigZag Conversion

2015-10-27

题目链接：ZigZag Conversion

题意：将给出的字符串按ZigZag形式排列，给出行数，求排列好后从左到右从上到下的文字顺序。

如(“PAYPALISHIRING”,4)输出”PINALSIGYAHRPI”

P     I     N

A L S I G

Y A H R

P I
可以$$(numRows-1)*2$$个字符一组，按字符的下标确定位置。

时间复杂度$$O(n)$$

 

public class Solution {
    public String convert(String s, int numRows) {
        if (numRows == 1)
            return s;
        int mod = (numRows - 1)  2;
        String ans = new String();
        int len = s.length();
        for (int i = 0; i < numRows; i++) {
            for (int j = i; j < len; j += mod) {
                int l = j, r = mod + j - 2  i;
                ans += s.charAt(l);
                if (r % mod > 0 && r != l && r < len) {
                    ans += s.charAt(r);
                }
            }
        }
        return ans;
    }
}
