LeetCode 15 3Sum AND 16 3Sum Closest AND 18 4Sum

2015-10-31

题目链接：3Sum

题意：给一个数组，找出所有不同的三元组(a,b,c)，满足a+b+c=0
 

对于K-Sum问题（$$\sum^K_i a_i = 0$$）都有通解

先对数组排序，再枚举$$K-2$$个数的和$$sum$$，再用二指针在$$O(n)$$时间求出剩下2个数$$(a,b)$$满足$$a+b=-sum$$，所以复杂度为$$O(n^{k-1})$$

对于去重，因为数组有序，所以可以直接跳过重复的数。同时注意左指针的起始位置。

时间复杂度$$O(n^2)$$

 

public class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> ans = new LinkedList<List<Integer>>();
        Arrays.sort(nums);
        int len = nums.length;
        for (int i = 0; i < len-2; ++i) {
            if (i > 0 && nums[i] == nums[i - 1])
                continue;
            int tag = -nums[i];
            int l = i+1, r = len - 1;
            while (l < r) {
                if (nums[l] + nums[r] == tag) {
                    List<Integer> node = new ArrayList<Integer>();
                    node.add(-tag);
                    node.add(nums[l]);
                    node.add(nums[r]);
                    ans.add(node);
                    while(l<r&&nums[l]==nums[l+1])l++;
                    while(l<r&&nums[r]==nums[r-1])r–;
                    l++;
                    r–;
                } else if (nums[l] + nums[r] < tag) {
                    l++;
                } else {
                    r–;
                }
            }
        }
        return ans;
    }
}

 

 

题目链接：3Sum Closest

题意：给一个数组，找出一个三元组(a,b,c)的和sum=a+b+c，满足sum最接近给出的target
 

方法和上一道题差不多，每次移动指针更新答案

时间复杂度$$O(n^2)$$

 

public class Solution {
    public int threeSumClosest(int[] nums, int target) {
        Arrays.sort(nums);
        int ansabs = 1 << 30;
        int ans = 0;
        int len = nums.length;
        for (int i = 0; i < len - 2; ++i) {
            int sum = target - nums[i];
            int l = i + 1, r = len - 1;
            while (l < r) {
                if (Math.abs(nums[l] + nums[r] - sum) < ansabs) {
                    ansabs = Math.abs(nums[l] + nums[r] - sum);
                    ans = nums[l] + nums[r] + nums[i];
                }
                if (nums[l] + nums[r] == sum) {
                    return target;
                } else if (nums[l] + nums[r] < sum) {
                    l++;
                } else {
                    r–;
                }
            }
        }
        return ans;
    }
}

 

题目链接：4Sum

题意：给一个数组和target，找出所有的一个四元组(a,b,c,d)满足a + b + c + d = target
 

同理，先对数组排序，再枚举两个数的和，然后用二指针处理剩下两个数，注意利用大小关系去重

时间复杂度$$O(n^3)$$

 

public class Solution {
    public List<List<Integer>> fourSum(int[] nums, int target) {
        List<List<Integer>> ans = new LinkedList<List<Integer>>();
        Arrays.sort(nums);

    for (int i = 0; i &lt; nums.length - 3; ++i) {
        if (i &gt; 0 &amp;&amp; nums[i] == nums[i - 1]) continue;

        for (int j = i + 1; j &lt; nums.length - 2; ++j) {
            if (j &gt; i + 1 &amp;&amp; nums[j] == nums[j - 1]) continue;

            int tag = target - nums[i] - nums[j];
            int l = j + 1, r = nums.length - 1;
            while (l &lt; r) {
                if (nums[l] + nums[r] == tag) {
                    List&lt;Integer&gt; node = new ArrayList&lt;Integer&gt;();
                    node.add(nums[i]);
                    node.add(nums[j]);
                    node.add(nums[l]);
                    node.add(nums[r]);
                    ans.add(node);
                    while (l &lt; r &amp;&amp; nums[l] == nums[l + 1]) l++;
                    while (l &lt; r &amp;&amp; nums[r] == nums[r - 1]) r--;
                    l++;
                    r--;
                } else if (nums[l] + nums[r] &lt; tag) {
                    l++;
                } else {
                    r--;
                }
            }
        }
    }
    return ans;
}

}
