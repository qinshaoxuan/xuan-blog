LeetCode 4 Median of Two Sorted Arrays

2015-10-26

题目链接：Median of Two Sorted Arrays

题意：给两个排好序的数组，求将两个数组合并之后的中位数。
 

可以把题目转换成求第k大的数。

采用二分的方法。从a,b两个数组的首部删去k/2个数（这里需要判断下边界，不能超出剩下数的数量），直到k==1或者某一个数组被删完。

判断a[k/2-1]与b[k/2-1]的大小

当a[k/2-1]<b[k/2-1]，那么a[0]~a[k/2-1]都是属于前k个数，可以直接删去，这时k变为$$\frac{k}{2}$$

当a[k/2-1]>b[k/2-1]，同理

当a[k/2-1]==b[k/2-1]，这时a数组可删的数+b数组可删的数=k，那么a[k/2-1]或者b[k/2-1]就是要找的中位数。

由此递归求解。

注意判断下总长度的奇偶，分类讨论下。

时间复杂度$$O(log(n+m))$$

 

public class Solution {
    public double find(int a[], int n, int b[], int m, int ha, int hb, int k) {
        // 参数为 (a数组,a数组长度,b数组,b数组长度,a数组头坐标,b数组头坐标,第k大)
        if ((n - ha) > (m - hb)) {
            return find(b, m, a, n, hb, ha, k);
        }
        if (ha == n) {
            return b[hb + k - 1];
        }
        if (k == 1) {
            return Math.min(a[ha], b[hb]);
        }
        int pa = Math.min(k / 2, n);// a数组删去的长度
        int pb = k - pa; // b数组删去的长度
        if (a[ha + pa - 1] < b[hb + pb - 1]) {
            return find(a, n, b, m, ha + pa, hb, k - pa);
        } else if (a[ha + pa - 1] > b[hb + pb - 1]) {
            return find(a, n, b, m, ha, hb + pb, k - pb);
        } else {
            return a[ha + pa - 1];
        }
    }

public double findMedianSortedArrays(int[] nums1, int[] nums2) {
    int len1 = nums1.length;
    int len2 = nums2.length;
    int len = len1 + len2;
    if (len % 2 == 1) {
        return find(nums1, len1, nums2, len2, 0, 0, len / 2 + 1);
    } else {
        return (find(nums1, len1, nums2, len2, 0, 0, len / 2) 
                + find(nums1, len1, nums2, len2, 0, 0, len / 2 + 1)) / 2;
    }
}

}
